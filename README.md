# კლინიკის მართვის სისტემა

React, Vite, Express და Prisma-ზე აგებული კლინიკის მართვის აპლიკაცია. პროექტი მოიცავს ავტორიზაციას, პაციენტების მართვას, სტაციონარის დანიშნულებებს, შაბლონებს და პაციენტის მონაცემების გარე რეესტრიდან მოძიებას.

## მოთხოვნები

- Node.js 20+
- npm

## გაშვება

1. დააყენეთ დამოკიდებულებები:
   `npm install`
2. შექმენით გარემოს ცვლადები:
   `cp .env.example .env`
3. გაუშვით აპლიკაცია:
   `npm run dev`
4. გახსენით:
   [http://localhost:3000](http://localhost:3000)

## საწყისი ავტორიზაცია

- მომხმარებელი: `admin`
- პაროლი: `admin123`

პირველ გაშვებაზე სისტემა ავტომატურად ამატებს ადმინისტრატორს SQLite ბაზაში, თუ ის ჯერ არ არსებობს.

## დამატებითი ბრძანებები

- `npm run lint` - TypeScript შემოწმება
- `npm run build` - ფრონტენდის build
- `npm start` - production რეჟიმში გაშვება build-ის შემდეგ
